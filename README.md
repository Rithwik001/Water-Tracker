💧 Water Tracker App (Flutter)

A clean and minimal Flutter application to track daily water intake and stay hydrated. The app provides a visual progress indicator and stores data locally so users never lose their progress.

🚀 Features
Track daily water intake in milliliters (ml)
Predefined buttons for quick logging (+100, +250, +500 ml)
Circular progress indicator with percentage completion
Persistent data storage using local device memory
Reset functionality to start fresh anytime
Modern UI with gradient background and glassmorphism effect

🛠️ Tech Stack
Flutter (UI Framework)
Dart (Programming Language)
SharedPreferences (Local Storage)

📱 How It Works
Users add water intake using quick buttons
Progress updates instantly in a circular indicator
Data is saved locally and restored when the app restarts
Intake is capped at the daily goal (2000 ml)
